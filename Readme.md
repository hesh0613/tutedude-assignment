# MERN Todo App

## Setup
1. Clone repo
2. Run `npm install`
3. Add `.env` file
4. Run `npm run dev`

## Tech Stack
- Node.js
- Express.js
- MongoDB
- React.js

## Features
- CRUD operations
- Search todos
- Status update
- API integration

## Challenges
- CORS issues → fixed using cors middleware
- Async errors → centralized error handler
